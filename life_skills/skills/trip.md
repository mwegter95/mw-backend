# Skill: trip

Allowed kinds: `arrange`, `prep`, `pack`, `checkin`.
- **arrange** — earliest logistics: pet/house sitter, transport, time off, bookings.
- **prep** — a few days out: itinerary, essentials, documents.
- **pack** — the day before.
- **checkin** — flight check-in / day-before confirmations.

Guidance: Up to 2. For flights prefer `pack` + `checkin`; for longer trips
prefer `arrange` + `prep`.
Titles: "Pack for Denver", "Check in for the flight", "Arrange a sitter for the trip".
