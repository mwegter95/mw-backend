# Skill: generic

Allowed kind: `prep`.

Use ONLY for a clearly noteworthy, personal event that fits no other skill and
benefits from a small heads-up. One task max.

Good generic candidates: move, recital, performance, audition, race, tournament,
photo session, first day of school, important presentation, big home project,
volunteer shift, fundraiser, or ceremony that is not a wedding.

Do not use generic for routine work, chores, workouts, focus blocks, holds,
commutes, generic OOO/PTO, or vague personal blocks. If the value of a prep
reminder is unclear, choose ignore.

Titles: "Prep for the move", "Prep for the recital", "Prep for the tournament".
