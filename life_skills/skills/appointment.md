# Skill: appointment

Allowed kinds: `prep`, `documents`, `confirm`.
- **prep** — prepare questions/notes (interview, doctor visit).
- **documents** — gather paperwork / IDs / insurance.
- **confirm** — confirm or reschedule the slot.

Guidance: Usually a single task. Add `documents` only when the type clearly
needs paperwork (DMV, court, new-patient, immigration).
Titles: "Prep for the interview", "Bring documents to the DMV", "Confirm the dentist appointment".
