# Skill: anniversary

Allowed kinds: `gift`, `plan`, `message`.
- **gift** тАФ a present, card, flowers, or thoughtful gesture.
- **plan** тАФ book/arrange something: dinner, date night, getaway, childcare, or
  a meaningful activity.
- **message** тАФ a simple note for someone else's anniversary.

Guidance:
- User's own/partner anniversary: prefer `plan` + `gift`.
- Someone else's anniversary: usually `message` only.
- If the event says "wedding anniversary", keep `anniversary`, not `wedding`.
- If the event implies travel/getaway for the anniversary, still use anniversary;
  the title can mention the plan.

Titles: "Plan our anniversary dinner", "Get an anniversary gift", "Wish Jenna a happy anniversary".
